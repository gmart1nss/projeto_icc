Hellfena Records

Projeto desenvolvido para a disciplina de Introdução à Ciência da Computação.

A proposta do trabalho foi desenvolver um catálogo temático de álbuns utilizando tecnologias web, aplicando conceitos fundamentais de estruturação de páginas, estilização, organização de conteúdo e interatividade.

Integrantes:

Gabriel Martins — desenvolvimento e implementação do projeto
Júlia Miqueline — desenvolvimento e implementação do projeto
Luana Sarto — apoio na construção visual e auxílio nos elementos de design

Tecnologias Utilizadas:

- HTML5
- CSS3
- JavaScript
  
Sobre o desenvolvimento:

O projeto foi construído com foco em criar uma experiência visual inspirada em páginas de catálogo de álbuns, com identidade própria para cada obra apresentada.

Durante o desenvolvimento foram utilizados recursos de assistência por IA para acelerar tarefas repetitivas e otimizar o tempo disponível para entrega. O uso dessas ferramentas ocorreu principalmente para apoio na construção das páginas individuais dos álbuns e na implementação de partes da página de contato em JavaScript, enquanto as decisões de estrutura, organização do projeto, identidade visual e integração final permaneceram sob responsabilidade da equipe.

Também foram realizados ajustes manuais de layout, composição visual, responsividade e refinamento estético ao longo do processo.

Funcionalidades:

- Catálogo de álbuns
- Sistema de pesquisa
- Filtro por categorias
- Ordenação de itens
- Páginas individuais para cada álbum
- Página de contato
- Interface com identidade visual inspirada no universo do metal
  
Considerações finais

Este projeto foi desenvolvido como atividade acadêmica da disciplina de Introdução à Ciência da Computação e serviu como prática de desenvolvimento front-end, organização de interface e integração entre HTML, CSS e JavaScript.
